# 12月开发更新

本次更新预计于12/25晚间发布。

## 玩法更新

### 新年特殊物品

新年快乐！2025即将结束，为了庆祝2026新年，我们重新加入了新年限定的烟火，tnt和雪球。

- `firework launch`在身边燃放随机烟花

- `firework get <1..5>`获取新年烟花

- `newyeartnt give`获取无伤害但是会造成更强击退效果的TNT

- `newyeardumplings`获取新年团子，有独特的击退效果。

### 常驻

加入了/gm命令用于切换游戏模式，/gm s切换到生存模式，/gm sc切换到旁观模式。

## 其他更新
